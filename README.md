# Mini Twitter Clone 

A simple **Twitter-like web application** built using **Node.js, Express.js, and EJS**.
This project demonstrates **RESTful API concepts and CRUD operations** where users can create, view, edit, and delete tweets.

---

## 📌 Features

* ✏️ Create a new tweet
* 👀 View all tweets
* 🔍 View tweet details
* 📝 Edit tweet content
* 🗑 Delete tweets
* 🖼 Add image URLs to tweets
* 📄 Clean UI using HTML, CSS, and EJS templates

---

## 🛠 Tech Stack

* **Node.js**
* **Express.js**
* **EJS (Embedded JavaScript Templates)**
* **HTML & CSS**
* **RESTful Routing**
* **Method-Override (PATCH & DELETE requests)**

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/mini-twitter-clone.git
```

### 2️⃣ Navigate to the project folder

```
cd mini-twitter-clone
```

### 3️⃣ Install dependencies

```
npm install
```

### 4️⃣ Run the server

```
node index.js
```

### 5️⃣ Open in browser

```
http://localhost:8080/twits
```
